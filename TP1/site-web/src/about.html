PolyPlay

Ce projet est un simple gestionnaire de listes de chansons (Playlists)

Vous pouvez générer des playlists composées des chansons disponibles sur le service

Vous pouvez utiliser les raccourcis suivants lorsqu'une chanson joue:

  • Espace : jouer/pause
  • N : prochaine chanson
  • P : chanson précédante
  • J : reculer de 5 secondes
  • L : avancer de 5 secondes
  • M : activer/fermer le son

Choisir une playlist à travers la Bibliothèque pour la faire jouer
